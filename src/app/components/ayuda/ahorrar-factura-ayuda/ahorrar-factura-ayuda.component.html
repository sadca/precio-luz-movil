Cerrar

Estos son los pasos a seguir para poder conseguir el ahorro que le proporcionaremos:

  1. Envienos sus facturas en el formulario para poder realizar el estudio. También puede enviar sus facturas a contacto@sadca.es en caso de tener alguna dificultad con el formulario.

  2. Nosotros contactaremos con usted por medio del correo electrónico que nos facilite o por el teléfono que nos indique. Le facilitaremos el estudio realizado para que pueda comprobar el ahorro estimado sobre sus facturas.

  3. Una vez decida que le interesa llevar a cabo este ahorro, nosotros le solicitaremos los datos restantes para formalizar el contrato y llevar a cabo su cambio para obtener dicho ahorro.

  4. Una vez formalizado el contrato, se lo enviaremos por correo para que lo valide y le enviaremos un SMS para que realice la firma de dicho contrato.

  5. ¡Contrato firmado!¡A disfrutar del ahorro!
